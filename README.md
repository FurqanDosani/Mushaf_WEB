# Mushaf_WEB

A web-based Qur’an (Mushaf) viewer built with Python, providing a clean and responsive interface for reading and navigating Surahs and Ayahs.

## Features
- Browse Qur’an by Surah and Ayah  
- Responsive and modern UI  
- Easy setup and local hosting  
- Planned: Search, Bookmarks, Dark Mode

## Installation
```bash
git clone https://github.com/FurqanDosani/Mushaf_WEB.git
cd Mushaf_WEB
pip install -r requirements.txt
python main.py
